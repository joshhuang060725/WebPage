# Personal Website

This is a basic static personal website.

## Files

- `index.html` - page content and SEO metadata
- `styles.css` - responsive layout and visual design
- `favicon.svg` - browser icon
- `og-image.svg` - social sharing preview image
- `robots.txt` - allows search engines to crawl the site

## Deploy

Deploy the folder as a static site.

For Cloudflare Pages:

- Build command: leave empty
- Build output directory: `/`

After connecting a real domain, update:

- `<title>`
- `meta description`
- `author`
- canonical URL
- JSON-LD `name`, `url`, and `sameAs`
- contact email
